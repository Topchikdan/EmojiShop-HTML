## О проекте

Учебный интернет-магазин смайликов на чистом JavaScript (ES-модули) с Vite и SCSS. Товары загружаются с внешнего API, есть поиск с debounce, корзина с сохранением между сессиями и обработка состояний загрузки, ошибки и пустого результата.

---

## Возможности

- Каталог товаров: карточки рендерятся динамически из данных API
- Поиск: фильтрация по названию и описанию в реальном времени с debounce
- Корзина: добавление/удаление товара, счётчик в шапке, данные сохраняются в `localStorage` между сессиями
- Состояния интерфейса: спиннер загрузки, сообщение об ошибке с кнопкой «Повторить», сообщение «ничего не найдено» для пустого поиска
- Уведомления (toast): короткое сообщение при добавлении/удалении товара из корзины
- Адаптивная вёрстка: сетка карточек и шапка перестраиваются под мобильные экраны

## Быстрый старт

Нужен [Node.js](https://nodejs.org/) версии 18 или новее.

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/Topchikdan/EmojiShop-HTML.git
cd EmojiShop-HTML

# 2. Установите зависимости
npm install

# 3. Запустите в режиме разработки
npm run dev
```

Откройте адрес из терминала (обычно http://localhost:5173).

## Структура проекта

```
smiles-shop/
├── index.html              # главная страница
├── package.json
├── vite.config.js
└── src/
    ├── js/
    │   ├── api.js           # запрос товаров к API
    │   ├── cart.js          # логика корзины (localStorage)
    │   ├── card.js          # отрисовка карточек товаров
    │   └── index.js         # точка входа: загрузка, поиск, события
    └── styles/
        ├── index.scss       # точка входа стилей
        ├── _variables.scss  # цвета, тени, радиусы, брейкпоинты
        ├── _fonts.scss      # подключение шрифтов Google Fonts
        ├── _global.scss     # сброс стилей + общие состояния (loading/error/empty/toast)
        └── blocks/
            ├── _header.scss
            ├── _shop.scss
            ├── _footer.scss
            └── _page.scss
```

## API

Товары загружаются с публичного тестового API на [mockapi.io](https://mockapi.io/). Ожидаемый формат одного товара:

```json
{
  "id": "1",
  "title": ":)",
  "description": "Веселый смайлик, наделяет улыбкой",
  "price": 200
}
```

Если API временно недоступен, приложение покажет сообщение об ошибке с кнопкой «Повторить».
