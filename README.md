# 😀 Smiles Shop — магазин смайликов

Учебный интернет-магазин «смайликов» на чистом JavaScript (ES-модули) с использованием Vite и SCSS. Товары загружаются с внешнего API, есть поиск, корзина с сохранением между сессиями и аккуратная обработка состояний загрузки/ошибки/пустого результата.

## ✨ Что есть в проекте

- **Каталог товаров** — карточки рендерятся динамически из данных API.
- **Поиск** — фильтрация по названию и описанию в реальном времени (с debounce).
- **Корзина** — добавление/удаление товара по кнопке, счётчик в шапке, данные сохраняются в `localStorage` и не пропадают при перезагрузке страницы.
- **Состояния интерфейса** — спиннер загрузки, сообщение об ошибке с кнопкой «Повторить», сообщение «ничего не найдено» для пустого поиска.
- **Уведомления (toast)** — короткое сообщение при добавлении/удалении товара из корзины.
- **Адаптивная вёрстка** — сетка карточек и шапка перестраиваются под мобильные экраны.

## 📁 Структура проекта

```
smiles-shop/
├── index.html              # главная страница
├── package.json
├── vite.config.js
└── src/
    ├── js/
    │   ├── api.js           # запрос товаров к API
    │   ├── cart.js          # логика корзины (localStorage)
    │   ├── card.js          # отрисовка карточек товаров
    │   └── index.js         # точка входа: загрузка, поиск, события
    └── styles/
        ├── index.scss       # точка входа стилей
        ├── _variables.scss  # цвета, тени, радиусы, брейкпоинты
        ├── _fonts.scss      # подключение шрифтов Google Fonts
        ├── _global.scss     # сброс стилей + общие состояния (loading/error/empty/toast)
        └── blocks/
            ├── _header.scss
            ├── _shop.scss
            ├── _footer.scss
            └── _page.scss
```

## 🚀 Запуск проекта

Понадобится [Node.js](https://nodejs.org/) версии 18 или новее.

# 1. Установить зависимости
npm install

# 2. Запустить дев-сервер с автообновлением
npm run dev

# 3. Собрать production-версию (папка dist/)
npm run build

# 4. Локально посмотреть собранную production-версию
npm run preview

После `npm run dev` Vite сам откроет браузер на `http://localhost:5173`.

## 🔌 API

Товары загружаются с публичного тестового API на [mockapi.io](https://mockapi.io/). Ожидаемый формат одного товара:

```json
{
  "id": "1",
  "title": ":)",
  "description": "Веселый смайлик, наделяет улыбкой",
  "price": 200
}
```

Если API временно недоступен, приложение покажет понятное сообщение об ошибке с кнопкой «Повторить» вместо пустого экрана.
